More Information

I’ve been working hard on this free tool that I believe can make a difference and save developers a couple of hours. This tool helps to convert Material Design 3 color schemes to .NET MAUI XAML resources. You can build the color scheme from the following pages: Material Foundation Theme Builder or Figma Material Theme Builder, and export the theme as JSON format, then paste the JSON in the left-box above and click the convert to XAML button.

Does it have a cost?

No, it is completely FREE.

Are you storing the data somewhere?

No, we don't save any data you converted with this tool. You can even disconnect from the internet and convert it offline, once the page is loaded.
